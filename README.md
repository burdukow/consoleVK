## **Информация:**<br>
Неофициальынй CLI клиент ВК, основанный на Python Requests.

## **Зависимости:**<br>
* Python 3<br>
* Requests<br>
* Аккаунт VK<br>

## **Установка:**
```shellsession
git clone https://github.com/burdukow/consoleVK.git
cd consoleVK
pip install -r requirements.txt
python main.py
```
